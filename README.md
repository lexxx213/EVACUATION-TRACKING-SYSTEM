# Evacuation Tracking System (EVTS)

A full-stack web application designed to monitor evacuation centers and evacuees in real-time.

# Features
- Admin Dashboard (monitor centers, reports, occupancy)
- Official Dashboard (manage evacuees per center)
- Live search and filtering
- Interactive map (Leaflet)
- Reports management system

## Tech Stack
- Frontend: Angular
- Backend: Node.js (Express)
- Database: MySQL

## How to Run
1. Run backend: node server.js
2. Run frontend: ng serve
3. Open http://localhost:4200
